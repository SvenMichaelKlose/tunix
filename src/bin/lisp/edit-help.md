TUNIX editor
============

| Navigation |                         |
|------------|-------------------------|
| Arrow keys | Move cursor.            |
| Ctrl+a     | Move to start of line.  |
| Ctrl+e     | Move to end of line.    |
| Ctrl+l     | Redraw screen.          |
|------------|-------------------------|
| Editing    |                         |
|------------|-------------------------|
| Ctrl+d     | Clear line.             |
|------------|-------------------------|
| Command    |                         |
|------------|-------------------------|
| Ctrl+k l   | Load file.              |
| Ctrl+k s   | Save file.              |
| Ctrl+k d   | List directory.         |
| Ctrl+k r   | Save and evaluate file. |
| Ctrl+k e   | Evaluate expression.    |
| Ctrl+k q   | Exit editor.            |
